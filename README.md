## 使用说明
```
选项：
  -c  并发数
  -n  请求总次数
  -a  请求持续时间，单位：秒，优先使用-n参数
  -m  HTTP method，默认：get
  -u  请求的url
  -d  post时的请求参数
  -f  从文件中获取可变参数信息
        get方式时，请求的地址是url和文件里面的行拼接而成
        post方式，文件里面是post的参数
  -t  超时时间，默认：1s
  -h  帮助
```

go run main.go -c 2 -n 200 -u "http://127.0.0.1:1234" -f get.txt
